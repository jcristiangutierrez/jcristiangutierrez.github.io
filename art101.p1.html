Drawing Application

For our first project in Art 101, we were tasked to develop our own drawing system, as well as our functions and arguments and to create some of our own brushes which we think best utilized our applications brushes. My inspiration for these brushes were optical illusions and trying to create something abstract. I wanted to create a brush that you can manipulate very easily, as well as making them transparent, so when crossing over another color they can create a new one. I really wanted to go for an abstract look on my drawings, I am pretty new to the style of abstract art so I tried to replicate my definition of abstract may or may not be correct. As for my code, I created a total of 5 functions, cGlowOprect(float xpos, float ypos, float width, float height), cGlowOpcircle(int xpos, int ypos, float height, float width), cGlowOpcircle2(int xpos, int ypos, float height, float width),cGlowOpcircle3(int xpos, int ypos, float height, float width),cGlowOpcircle(int xpos, int ypos, float height, float width), cGeye(int xpos, int ypos, float width, float height), and finally GrainboLine(color c, int ax, int ay, int px, int py), I did make some drawings using this brush, but I just did not find them as powerful as the other drawings.My favorite would have to be the the brush withe the transparent rectangles because I was able to create these futuristic looking type snow mountain which reminded me of cover art for an old arcade game.
